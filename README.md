## แนวข้อสอบ CF
**น่าจะจารย์สุเมธ**
- เติมคำ เป็นเลือก ๆ ถูกทุกอันจะให้คะแนน
- bonus สำฟรับคนเสร็จเร็ว ก่อน 90นาทีแรก จะได้ password ตอนสุดท้ายเพื่อไปเอาคะแนนโบนัส 3 คะแนน  
(25+3คะแนน)
- โบนัสจะให้ทำอีกครึ่งชม

**อาจารย์ประพันธ์**
- ตอนแรกที่เคยบอกว่าจะเป็น 60 ข้อ ดึง15ข้อไปทำป็นวิธีทำ
- เปลี่ยนเป็นช้อย 75 ข้อ แล้วก็มี bonusเป็นวิธีทำ 3 คะแนน
- เกี่ยวกับ docker ไม่มีคำสั่งให้ ต้องจำเอง (แต่เขาบอกอันนี้มา ไม่รู้หมายถึงว่าไปเปิดในห้องได้ไหม) ปกจิพิมพ์ docker ใน cmd แล้วจะมี คำสั่งขึ้นมา (docker/ docker run --help ไรงี้) ถ้าใช้ docker desktop ให้กดดู container แล้วอะไรต่อไม่รู้ ฟังไม่ทัน
